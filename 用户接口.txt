1. 用户登录
   请求地址: http://{host}/users
   请求方式: POST
   请求参数: username -> 用户名
             password -> 密码
   返回参数: {token: "用户token值"}
   
2. 用户修改资料
   请求地址: http://{host}/users/{token}
   请求方式: POST
   请求参数: token -> 用户token信息
             name -> 用户名字
             icon -> 用户头像
             signature -> 签名
             gender -> 性别
             identity -> 身份 // 1，私教 2.健身爱好者 3.商家
   返回参数: {code: 1}
   
3. 用户登出
   请求地址: http://{host}/users/{token}
   请求方式: DELETE
   请求参数: token -> 用户token信息
   返回参数: {code: 1}
   
4. 用户信息
   请求地址: http://{host}/users/{token}
   请求方式: GET
   请求参数: token -> 用户token信息
   返回参数: {
                name: "用户名字",
                icon: "用户头像",
                signature: "签名",
                gender: 性别
                identity: 身份
             }
